# Tutorial de Desarrollo de Odoo 16.0

## Descripción general

¡Bienvenido al tutorial de Odoo 16.0! Si has llegado a esta página, significa que te interesa desarrollar módulos de Odoo con herramientas, scripts, plugins, optimización de código
y técnicas avanzadas.

El objetivo de este tutorial es que te familiarices con los aspectos más importantes del framework de desarrollo de Odoo 
con herramientos, documentación, scripts, plugins para optimizar el tiempo de desarrollo y obtener un producto de calidad. 
Los capítulos deben seguirse en el orden indicado, ya que abarcan el desarrollo de una nueva aplicación de Odoo desde cero de forma incremental. En otras palabras, cada capítulo depende del anterior.
Cubriremos todos los conceptos básicos que necesitas para poder crear aplicaciones
óptimas, escalables y reutilizables en Odoo.

Antes de empezar con la configuración del entorno de desarrollo, asegúrate de tener instalado los siguientes requisitos previos: Python 3, Git, PostgreSQL y VSCode. Si aún no los tienes, puedes seguir los siguientes enlaces para instalarlos:

Python 3: python.org/downloads

Git: git-scm.com/downloads

PostgreSQL: postgresql.org/download

VSCode: code.visualstudio.com/download
